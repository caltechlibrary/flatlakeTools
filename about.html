About this software

flatlakeTools 0.0.2

working proof of concept supporting RSS and Markdown generations as well as copying a CommonMark document into the blog directory string after validation.

Authors

  • R. S. Doiel Doiel, <orcid.org/0000-0003-0900-6903>

This is an experimental proof of concept set of tools for working with the FlatLake API generated by CloudCannon's Flat. The goal is to easily generate RSS 2.0 feeds from the JSON API as well as generated index blog page(s).
